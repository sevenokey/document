经验技能可复用skills

Anthropic 的 Skills（技能）是一套面向任务、可复用、可组合的“能力包”体系，用来把完成某件事所需的指令、脚本和资源封装成一个文件夹，让 Claude 只在需要时才完整加载，从而在几乎不额外占用上下文的前提下，把大模型临时“训练”成领域专家。它的核心特点可以概括为 5 句话：
1. 按需加载：平时只保留 YAML 元数据，任务匹配才把完整 Skill 读入，省 token。
2. 文件即技能：一个文件夹 + SKILL.md（含步骤、脚本、资源）就能定义技能，可版本管理、回滚、审计。
3. 可组合/可移植：多技能像乐高一样拼装，同一套技能在 Claude.ai、Claude Code、API、Agent SDK 间无缝迁移。
4. 代码级执行：Skill 里可放 Python、Shell 等脚本，Claude 在沙箱内直接运行，实现“命令行能做的事都能自动化”。
5. 生态开放：官方已在 GitHub 开源 16 个示例（PDF、Excel、PPT、品牌合规等），开发者可按模板快速自建或上架市场。
典型用法举例
* 个人：输入 /code-review 自动按团队规范检查代码；/git-commit "message" 一键完成 add/commit/push。
* 企业：把品牌指南做成 Skill，Claude 生成的任何对外文案、PPT、海报都自动符合 VI 标准；把 Notion/Box 内容拖进来即可按模板输出成公司统一格式的报告。
* 开发者：通过 API 给 Agent 动态下发 Skill，实现“今天写爬虫、明天跑数据分析”而不用改主程序。
一句话总结：Skills 让 Claude 从“对话助手”升级为“通用计算机 Agent”，把“怎么做”沉淀成可分享、可演进、可安全执行的文件级知识，解决了以前 Prompt 一次性、难维护、难复用的问题

存储位置：
优先级顺序： OpenSkills 按优先级顺序搜索 4 个位置：
./.agent/skills/ (项目级)
~/.agent/skills/ (全局)
./.claude/skills/ (项目级)
~/.claude/skills/ (全局)

命令:
openskills install <source> [options]  # Install from GitHub, local path, or private repo
openskills sync [-y] [-o <path>]       # Update AGENTS.md (or custom output)
openskills list                        # Show installed skills
openskills read <name>                 # Load skill (for agents)
openskills manage                      # Remove skills (interactive)
openskills remove <name>               # Remove specific skill

--global— 全局安装~/.claude/skills（默认：项目安装）
--universal— 安装到.agent/skills/（.claude/skills/高级）
-y, --yes— 跳过所有提示，包括覆盖（针对脚本/CI）
-o, --output <path>— 同步的自定义输出文件（默认值AGENTS.md：）

快速入门
1. 安装
npm i -g openskills
2. 安装技能
# Install from Anthropic's marketplace (interactive selection, default: project)
openskills install anthropics/skills

# Or install from any GitHub repo
openskills install your-org/custom-skills
3. 同步到 AGENTS.md
注意：必须预先存在 AGENTS.md 文件，同步才能更新。

openskills sync


如何使用它：

用户请求：“从这个PDF文件中提取数据”
扫描<available_skills>→ 发现“pdf”技能
代理调用：Bash("openskills read pdf")
SKILL.md 的内容会输出到代理的上下文中。
代理人按照指示完成任务

安装模式
默认值（推荐）：

openskills install anthropics/skills
# → Installs to ./.claude/skills (project, gitignored)
全局安装：

openskills install anthropics/skills --global
# → Installs to ~/.claude/skills (shared across projects)
通用模式（高级）：

openskills install anthropics/skills --universal
# → Installs to ./.agent/skills (for Claude Code + other agents)
